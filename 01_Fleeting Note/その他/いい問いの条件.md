#DR 
以下は、安斎勇樹・塩瀬隆之『問いのデザイン―創造的対話のファシリテーション』で示されているエッセンスを整理した「いい問い（良質な問い）の条件」です。箇条書きの数字・用語は原著のキーメッセージを噛み砕いて再構成したもので、本文のニュアンスを損なわない範囲で簡潔にまとめています。

# **1. まず押さえたい「３つのコア条件」**

|**条件**|**ねらい**|**キーワード**|
|---|---|---|
|① 問いがいがある|簡単に正解が出ず、多様な視点を引き出す。「はい／いいえ」で終わらないオープン‐エンド。|探究心・生成性|
|② 自分ごととして捉えられる|参加者の経験・関心に結び付き、主体性（当事者意識）を呼び起こす。|内発的動機づけ|
|③ 行動につながる可能性がある|思考や対話が、次のステップ・変化・実装へ接続しやすい。|アクション・変容|

この３条件は、**問いが「議論を回すだけで終わらず、探究と実装を往復させるエンジンになるか」**を測る最小チェックリストです。

# **2. 良い問いが備える「７つの基本性質」**

原著では、問いが場にもたらす働きを７つの性質として整理しています。言い換えれば、**以下を一つでも多く満たすほどその問いは“質が高い”**と評価できる、という指標です。

1. 答えを方向付ける – 問いの立て方次第で導かれる答えは変わる
2. 思考と感情を同時に刺激する – 頭も心も動かす
3. コミュニケーションを誘発する – 集団に対話を起こす触媒
4. 個人の内省を促す – 自分の前提・価値観を再点検させる
5. 関係性を再構築する – 対話過程でチームの関係が変わる
6. 創造的対話のトリガーになる – アイデア創発を駆動
7. 新たな問いを生む – 探究が連鎖し、学習が継続する

チェック方法

- その問いを投げた後、参加者の表情と言葉に「驚き」「わくわく」「沈黙と反芻」が混ざるか？
- 対話中に “問い返し” や “問いの言い換え” が自然発生するか？
- 終わった後に「じゃあ次に●●を試したい」が出るか？

# **3. “深さ”をデザインする３変数**

問うテーマが同じでも**深さ（どれだけ腰を据えて考える必要があるか）**で問いの質は大きく変わります。深さは次の３つの変数で測れると解説されています。

|**変数**|**浅い問い**|**深い問い**|
|---|---|---|
|① 関わる視点・視座の数|単一視点で完結（例：自分の朝食は？）|多層視点を要する（例：持続可能な朝食とは？）|
|② 答えの多様性|ほぼ１つ（または少数）|人数分だけ異なる／無数に分岐|
|③ 回答に必要な時間・労力|数秒〜数分で済む|継続的調査・議論が必要|

実務ヒント

- アイスブレイクでは“浅い問い → 中くらいの問い”と徐々に深度を上げる
- ハードデッドラインがあるワークショップでは「深さ★２」程度に抑え、残りは宿題化する

# **4. つくるときのステップ（簡易版）**

1. 目的を言語化
    - 何を明らかにしたいのか、決めたいのか、創りたいのか

2. ７性質＋３条件のラフ評価
    - ★を付けて不足があれば書き換える

3. 深さの調整
    - 時間・メンバー構成・場のフェーズと照合して「ちょうど良い歯ごたえ」にする

4. 言葉を磨く
    - 主語・動詞・形容詞を具体→抽象へ／抽象→具体へ往復しながら“刺さる一文”に仕上げる

5. リハーサル／パイロット
    - 小人数で試し投げし、反応・沈黙時間・盛り上がり度を観察して微調整

# **5. 逆にやりがちな「課題設定５つの罠」（抜粋）**

自分本位／自己目的化／前提未検証／時間感覚不整合／議論のスコープ過大 など、問い以前の段階で失敗を招く典型パターンも紹介されています。

これらは 「問いがいはあるけれど行き止まり」 という落とし穴になりやすいので、上記ステップ②③で必ずチェックしましょう。

# **6. ミニ事例（ビフォー／アフター）**

|**ビフォー（ありがちな問い）**|**アフター（良い問いへ再設計）**|**着目した条件**|
|---|---|---|
|「売上を伸ばすには？」|「私たちの顧客が“また来たい”と感じる体験とは？」|①問いがい、②自分ごと、③行動|
|「働きがいのある職場にする方法は？」|「明日チームの誰もが“誇らしい一日だった”と語れる瞬間を生むには？」|①＋②＋深さ調整|

**まとめ**

- ３コア条件で“問いとして成立するか”をまず確認
- ７基本性質で“問いが生む価値”を高める
- 深さ３変数で“時間・文脈に合った歯ごたえ”を設計
- ステップを踏んで言葉を磨き、罠を避ければ 「思考と行動を同時に動かす問い」 が生まれる

これらを意識すると、ワークショップでも会議でも学習支援でも、場が「答え合わせ」から「共創の探究」へと自然にシフトします。ぜひ次のファシリテーション・議論設計で試してみてください。
