# みんなで作る経済ニュース”Newspicks”

Created: 2023年11月23日 16:59
Tags: UX, コミュニティ, 所属感アシスト, 貢献の余白

- 貢献の余白
    - 人は完璧なものより、「あ、わたしならこうするな」とか「ここはわからないから自分で考えないといけないか」など、わきの甘さが多少ある物の方に興味を惹かれ、あれこれ思考を巡らし始める。
    - ただし、余白が大きすぎるお余白の意味さえ薄れてしまう
    - どこにどんな余白を設けるのかによって、顧客のスイッチをいれることができる
    - 余白が出る事で、自分が関与する機会となり、存在価値、居場所、愛着になる
    - プロセス参加との違いは、未完成部分があったとしてもプロセスが感慮している点
- 経済ニュースに自分のコメントをつけたり、他者のコメントにいいねをつけたりすることもできる
    - Knotworkingに似ていると思ったけど、別に友達になったりはしないのか。コミュニケーションが限られているし、リアルであったりしない。それ自体が楽しめるアイコンになっていないだろうから。
