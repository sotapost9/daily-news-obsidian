# ニーズは増産できるがジョブは開発できる

**ニーズとジョブの決定的な違い：**

- **「ニーズのある商品を増産する」はできる**が、「ニーズのある商品を開発する」はできない
- **ニーズは顕在化された購買事例が対象**。明確なものにしか適用できない
- **ジョブは新しいイノベーションのための羅針盤**になる

## ニーズとジョブの関係は「天気と天気図」

- その場その場での天気（ニーズ）はピンポイントに把握できる
- 将来の天気を予測するには天気図（ジョブ）の力が必要

**ニーズが生まれる条件：**
- 顧客のジョブが強い
- かつ商品・サービスがジョブを解決する
→ ニーズがある

ジョブが弱かったり、ジョブの解決に不十分だったりすると、あまりニーズは生まれない。
