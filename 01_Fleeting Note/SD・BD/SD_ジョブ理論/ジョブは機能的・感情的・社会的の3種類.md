# ジョブは機能的・感情的・社会的の3種類

ジョブには3つのタイプが存在する：

1. **機能的ジョブ**：単純に何かをどうにかしたいというもの
   - 例：会社まで移動したい、空腹を満たしたい
   - 「どう為し遂げるか」という観点で評価される

2. **感情的ジョブ**：気分や感情に関わるもの
   - 例：「退屈をしのぎたい」「優しい父親の気分を味わう」「おしゃれな気分を味わいたい」
   - 「どう感じるか」という観点で評価される

3. **社会的ジョブ**：他人からどう見られたいかに関わるもの
   - 例：「成功している」「センスがよい」と思われたい
   - 「どう見られるか」という観点で評価される

**重要：**売り手になると機能的な側面ばかりを気にしてしまうが、感情的・社会的ジョブは見落としがち。先進国ではほとんどの機能的ジョブは既に解決されているため、感情的・社会的ジョブに着目することが重要。
