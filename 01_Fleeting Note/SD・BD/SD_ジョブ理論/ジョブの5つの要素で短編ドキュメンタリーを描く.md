# ジョブの5つの要素で短編ドキュメンタリーを描く

ジョブ理論を活用する際は、**短編ドキュメンタリーをイメージする**ことが有効。以下の5つのステップで構成する：

1. **その人が成し遂げようとしている進歩は何か**
   - 例：仕事やプライベートで印象のいい笑顔を浮かべたい
2. **苦心している状況は何か**
   - 例：年二回歯医者に行っているのに思ってるより歯が白くならない
3. **進歩を成し遂げるのを阻む障害は何か**
   - 例：歯を白くする歯磨き粉をいくつか試したが、効果はなく、誇大広告だった
4. **不完全な解決策で我慢し、埋め合わせの行動をとっていないか**
   - 例：一時凌ぎのために高価なマウスピースを一日中はめないといけないし、ヒリヒリする
5. **その人にとって、よりよい解決策をもたらす品質の定義は何か。また引き換えにしてもいいと思うものは何か**
   - 例：専門治療によるホワイトニングを受けたい。ただし費用も手間もかけたくない

この5要素はいずれも重要な文脈であり、ジョブ理論はこれらのパックとも言える。
