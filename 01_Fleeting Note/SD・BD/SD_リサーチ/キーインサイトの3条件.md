# キーインサイトの3条件

優れたキーインサイトには3つの条件がある：

## 1. バリュープロポジションと満たし満たされの関係になっていること

インサイトと提案価値がセットで示せる状態。キーインサイトとバリュープロポジションはよく使う典型パターン。

## 2. 不満、または未充足欲求のエモーションと、他の要素の組み合わせで構成されていること

インサイトの3つのタイプ：
- **価値**：ある対象へのポジティブな心理
- **不満**：ある対象へのネガティブな心理
- **未充足欲求**：求めているがまだ充たされていない心理

キーインサイトは不満か未充足欲求である必要がある。

## 3. シーン、背景、ドライバーのいずれか一つ以上であること

感情だけではなく、客観的な事実（シーン、背景、ドライバー）と組み合わせる必要がある。

## エンジェルインサイトとデビルインサイト

人には必ず善と悪の部分が両面ある。善だけ考えていては綺麗事になってしまうので両面から考えることが必要。

**例：**マックのユーザーにはヘルシーを求めるリサーチも多かったが、実際にはビックマックにかぶりつきたいというデビルインサイトの方が強かった。
