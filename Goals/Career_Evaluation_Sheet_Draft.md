# キャリア・評価シート向け分析資料 (2026-02-19)

上司への評価シート記入依頼・自己評価に向けた、Mid-Long Term（5年後など）の「To-Be（あるべき姿）」と「As-Is（現状）」の整理です。

---

## 1. 目指す人材像・キャリア (Ideal Self/Career)

### To-Be (中長期：5年後〜)
**「日本一のEmpathic Designer / 社会課題解決を実装するπ型人材」**
*   **コア・アイデンティティ**: デザイナーと異分野（ビジネス、エンジニアリング、福祉、哲学など）の間に立ち、翻訳者として機能する「π（パイ）型人材」。
*   **専門性**: サービスデザイン × デザインリサーチ（特にインクルーシブリサーチ・当事者研究）。
*   **ありたい姿**:
    *   「効率とエンパシー」「資本主義と倫理」のような二項対立を乗り越える「闘争するサービスデザイン」の実践者。
    *   センシティブなテーマ（メンタルヘルス、貧困、死生観など）において、当事者に深く寄り添いながらビジネスとして成立させる社会起業家・デザイナー。
    *   社内外で「Empathyといえば山中」と想起される存在（社内最高倫理責任者のような立ち位置）。

### As-Is (現在：2026年初頭)
**「情熱と行動力はあるが、足腰（ロジック・型）の強化が必要なフェーズ」**
*   **ポジション**: アナリスト（シニアアナリストへの昇格を目指す段階）。
*   **強み**:
    *   高い熱量とコミットメント（「プライドが高い」と評されるほどの意思）。
    *   建築・空間設計などのバックグラウンドを活かした独自の視点。
    *   UnD（Universal Design）やI&Dコミュニティへの自発的な貢献とリーダーシップ。
*   **課題**:
    *   「やりたいこと」への思いが先行し、組織として必要な「泥臭い仕事」や「型」への適応が発展途上。
    *   クライアントやMDと対等に渡り合うための「ロジカルシンキング」「共通言語」の不足。

## 2. 経験していきたいこと (Experience)

### To-Be (中長期)
**「異なる領域を横断し、社会的に重い課題（Pain）を解決する経験」**
*   **領域**:
    *   病院経営の改革、企業のメンタルヘルス改革、就労支援のUX改革など、医療・福祉・ヘルスケア領域。
    *   当事者（マイノリティ、患者）参画型の共創プロジェクト。
*   **役割**:
    *   プロジェクト全体をリードするPM/デザインリード。
    *   異なる職能（エンジニア、ビジネスサイド）を繋ぐ「翻訳者」としてのハブ役。
*   **コミュニティ・発信**:
    *   UnDなどのコミュニティ主催を通じた「場づくり」の経験。
    *   「Empathic Design」の第一人者としての登壇・執筆・ラジオ発信。

### As-Is (現在)
**「PJリーダーとしての『型』を習得し、守備範囲を広げている段階」**
*   **実績**:
    *   Pokemon PJでのWS設計・ファシリテーション・進行リード（評価：Good）。
    *   WEINS PJでの建築知見を活かした信頼獲得。
    *   Obsidian勉強会などの自主的なナレッジシェア。
*   **直近の経験目標**:
    *   「ミニマネージャー」として、特定のタスク領域（リサーチ、WSなど）を完結してリードする経験。
    *   メンタルヘルス・マネジメント検定（II種/I種）の取得を通じた、専門領域の知識獲得。

## 3. 身につけたいケイパビリティ (Capabilities)

### To-Be (中長期)
**「AIに代替されない人間力 × 異分野をつなぐ翻訳言語（スキル）」**
*   **Human Skills (AI代替不可)**:
    *   **倫理的判断力 (Ethics)**: 「何をすべきでないか」を判断する羅針盤。
    *   **深い共感力 (Empathy)**: N1の痛みを想像し、憑依する力。
    *   **責任を負う力**: 決断し、結果を引き受ける覚悟。
*   **Hard Skills (翻訳言語)**:
    *   **インクルーシブリサーチ**: センシティブな対象者へのインタビュー・観察技術。
    *   **ビジネスデザイン**: 課題を「お金に変える（事業化する）」ロジックと戦略眼。
    *   **ストーリーテリング**: 複雑な事象を感情に訴えかける物語に変換する力。

### As-Is (現在)
**「独自の『感性』をクライアントに届けるための『論理』の習得」**
*   **習得中/課題**:
    *   **ロジカルシンキング**: 2026年の最重要テーマ。クライアントフェーシングに耐えうる論理構成力。
    *   **ドキュメンテーション**: スライド作成、資料作成の「型」とスピード。
    *   **作業の効率化**: クオリティとスピードを両立させるための業務プロセスの型化。
*   **保有**:
    *   デザインへの熱意、建築的思考、Obsidianを活用したナレッジ管理能力。

## 4. 目指す姿に向けたPJ (Projects)

### To-Be (中長期)
**「『闘争するサービスデザイン』を体現する、高難易度かつ社会的意義の高いPJ」**
*   **テーマ**:
    *   メンタルヘルス、障害者支援、貧困、D&Iなど、資本主義の論理だけでは解決しづらいテーマ。
    *   「ケア」と「ビジネス」の両立が求められるPJ。
*   **スタンス**:
    *   単なる受注作業ではなく、クライアントと共に社会課題の根源に向き合うパートナーシップ型のPJ。
    *   「裂け目（組織やサービスの分断）」を埋めるような、統合的なPJ。

### As-Is (現在)
**「ロジックと信頼を積み上げ、『やりたい案件』を引き寄せるための足場固めPJ」**
*   **現状のPJ**:
    *   大手クライアントのサービス改善・構想案件（Pokemon, WEINSなど）を通じ、基礎的なコンサルティングワークと信頼関係構築を学んでいる。
*   **意識すべきこと**:
    *   自分の「やりたいこと（Empathy/社会課題）」を主張するだけでなく、目の前のPJで圧倒的な成果（ロジック・効率・雑務含む）を出し、組織とクライアントからの「信頼貯金」を貯める時期。
    *   今のPJの中で「Empathy」や「倫理」の視点をどう滑り込ませるか、という小さな実験の実践。
