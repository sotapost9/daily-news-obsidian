# 読書会の成果目標設計｜ジョブ理論アプローチ

作成日：2025-10-22

---

## 🤔 「参加人数」だけでいいのか？

参加人数は確かに分かりやすい指標ですが、**ジョブ理論の視点では不十分**です。

### 参加人数だけを追うリスク

❌ **量を追って質が下がる**
- 誰でもいいから集めると、コアな価値観が薄まる
- 「敷居を下げすぎる」と本質的な対話が生まれにくい
- 初参加者ばかりで深い議論ができない

❌ **本質的な価値を測れない**
- 10人来ても誰も満足していなければ意味がない
- 参加者が「雇った」ジョブが達成されたかが重要
- リピートしない参加者を集め続けるのは疲弊する

❌ **長期的な成長に繋がらない**
- コミュニティが育たない
- 口コミが生まれない
- ブランドが確立しない

---

## 🎯 ジョブ理論に基づく成果目標の設計

### 大前提：読書会の「コアジョブ」を再確認

```
「倫理的な視点で本を語り合い、
利他的な知的コミュニティで
安心して自分を表現したい」
```

この**コアジョブが達成されているか**を測ることが最重要です。

---

## 📊 4階層の成果指標モデル

### レイヤー1：ジョブ達成指標（最重要）
**「参加者のジョブは達成されたか？」**

#### 指標1-1：進歩達成率（Job Success Rate）
```
アンケート質問：
「今日の読書会で、あなたが求めていた体験はできましたか？」

測定：
- できた：80%以上目標
- ある程度できた：15%
- できなかった：5%未満目標
```

#### 指標1-2：ジョブ別達成率
```
「今日の読書会で達成できたこと」（複数選択）
□ 新しい視点に出会えた → 目標80%以上
□ 本について深く対話できた → 目標70%以上
□ 安心して自分の考えを話せた → 目標90%以上
□ 同じ価値観の人と繋がれた → 目標60%以上
□ 倫理的なテーマを深められた → 目標70%以上
□ 読みたい本が見つかった → 目標50%以上
```

#### 指標1-3：Time to First Job（初回進歩までの時間）
```
「参加して何分後に『楽しい』『来てよかった』と感じましたか？」

目標：開始10分以内（近況共有の時点）に60%以上が達成
```

---

### レイヤー2：関係性指標（コミュニティの質）
**「繋がりは生まれているか？」**

#### 指標2-1：NPS（Net Promoter Score）
```
「友人に読書会を薦める可能性は？」（0-10点）

計算：
- 推奨者（9-10点）の割合 - 批判者（0-6点）の割合

目標：+30以上（業界平均は+10程度）
理想：+50以上（世界トップクラス）
```

#### 指標2-2：リピート率
```
「また参加したいですか？」

目標：
- 「はい」が80%以上
- 実際のリピート率が50%以上
```

#### 指標2-3：参加者同士の繋がり率
```
「読書会の外でも参加者と交流しましたか？」

測定：
- Slack等での交流
- 本の貸し借り
- 1on1での対話
- SNSでの相互フォロー

目標：30%以上が何らかの繋がりを持つ
```

---

### レイヤー3：行動変容指標（影響の質）
**「読書会が参加者の行動を変えたか？」**

#### 指標3-1：読書行動の変化
```
「読書会参加後、読書習慣に変化はありましたか？」

測定：
□ 紹介された本を読んだ
□ 関連テーマの本を探した
□ 読書量が増えた
□ 読書の視点が変わった

目標：60%以上が何らかの変化
```

#### 指標3-2：思考・行動の変化
```
「読書会での対話が、日常の思考や行動に影響を与えましたか？」

測定：
□ 倫理的な消費を意識するようになった
□ 資本主義について考えるようになった
□ 他者との対話の質が変わった
□ 自分の価値観が明確になった

目標：40%以上が何らかの影響
```

#### 指標3-3：アウトプット創出率
```
「読書会の内容をアウトプットしましたか？」

測定：
□ SNSで感想を投稿した
□ noteに記事を書いた
□ 友人や同僚に話した
□ 仕事に活かした

目標：50%以上が何らかのアウトプット
```

---

### レイヤー4：運営継続指標（持続可能性）
**「読書会は続けられているか？」**

#### 指標4-1：参加者数
```
目標：
- 1回あたり5-10名（質の高い対話が可能な人数）
- 初参加者：2-3名/回（新陳代謝）
- リピーター：3-5名/回（コア形成）
```

#### 指標4-2：開催頻度
```
目標：
- 月1回以上の開催
- 年間10回以上
- 3ヶ月連続中断なし
```

#### 指標4-3：運営負荷
```
測定：
「準備・運営にかかる時間は適切ですか？」

目標：
- 準備：2時間以内/回
- 運営：1-1.5時間/回
- フォロー：1時間以内/回
- 合計：4.5時間以内/回

持続可能性を重視
```

## 🎯 フェーズ別の重点指標

### フェーズ1：立ち上げ期（1-3ヶ月）
**目標：「安心して参加できる場」を作る**

**重点指標：**
1. ✅ 安心して話せた率：90%以上
2. ✅ また参加したい率：80%以上
3. ✅ 参加者数：5名以上で安定

**この時期は質を最優先**
- 少人数でもいいから深い対話
- リピーターを3-5名作る
- 型を確立する

---

### フェーズ2：成長期（4-6ヶ月）
**目標：「コミュニティ」を育てる**

**重点指標：**
1. ✅ リピート率：50%以上
2. ✅ NPS：+30以上
3. ✅ 参加者同士の繋がり率：30%以上
4. ✅ 開催継続：月1回以上

**この時期はコミュニティ形成**
- Slack等で繋がる
- 参加者同士の交流を促進
- 口コミで広がり始める

---

### フェーズ3：確立期（7-12ヶ月）
**目標：「ブランド」を確立する**

**重点指標：**
1. ✅ NPS：+50以上
2. ✅ 行動変容率：40%以上
3. ✅ アウトプット創出率：50%以上
4. ✅ 年間10回以上開催達成
5. ✅ Song内外での認知向上

**この時期は影響力拡大**
- コンテンツ化（note、Podcast等）
- 他コミュニティとのコラボ
- 「Songの倫理読書会」としてのブランド確立

---

### フェーズ4：発展期（13ヶ月〜）
**目標：「ムーブメント」を作る**

**重点指標：**
1. ✅ コミュニティメンバー：30名以上
2. ✅ 月間アクティブ率：40%以上
3. ✅ メディア展開（YouTube/Podcast）
4. ✅ 外部からの引き合い
5. ✅ 共同運営者の育成

**この時期は拡張と委譲**
- チャンネル開設
- 企業向けワークショップ
- 読書会の型を他に展開

## 💡 よくある質問

### Q1: 参加者数が少ない時はどう考えればいい？

**A:** ジョブ達成率が高ければ問題なし。

- 5名でも全員が満足して、深い対話ができていればOK
- むしろ少人数の方が深い対話が生まれやすい
- リピート率が高ければ、徐々に口コミで広がる

**重要なのは「参加者数」より「参加者の満足度×リピート率」**

---

### Q2: 指標が多すぎて測定が大変では？

**A:** フェーズごとに重点指標を絞る。

**立ち上げ期（最初の3ヶ月）は3つだけ：**
1. 安心して話せた率
2. また参加したい率
3. 参加者数

これだけ測定して、目標達成できたら次のフェーズへ。

---

### Q3: NPSって何？どう計算するの？

**A:** Net Promoter Score = 顧客ロイヤルティの指標

**質問：**
「友人に読書会を薦める可能性は？」（0-10点）

**計算：**
- 推奨者（9-10点）の割合 - 批判者（0-6点）の割合
- 例：推奨者60%、批判者10% → NPS = +50

**目安：**
- +30以上：優秀
- +50以上：世界トップクラス
- +70以上：伝説級

---

### Q4: 目標未達成の場合はどうする？

**A:** 原因を特定して改善する。

**ジョブ達成率が低い場合：**
- 参加者にヒアリング「何が物足りなかった？」
- フローを見直す
- テーマを変える

**リピート率が低い場合：**
- 参加後のフォローを強化
- コミュニティ施策を追加
- 次回のテーマ設定を工夫

**重要：** 完璧を求めず、PDCAを回し続ける

```

```

## ✨ 最後に：数字の奴隷にならないために

指標は大切ですが、**数字を追うことが目的ではありません。**

### 本当に大切なこと

**✅ 参加者一人ひとりの笑顔**
- 数字に現れない満足感
- 対話の中で生まれる気づき
- 繋がりの温かさ

**✅ あなた自身が楽しめているか**
- 義務感で続けても持続しない
- 主催者が楽しんでいる場は良い場になる
- 数字よりも対話の質を大切に

**✅ コミュニティの文化**
- 利他、共感、安心
- この文化が守られていれば、数字は後からついてくる

---

### 推奨：バランスの取れた目標設定

**60%：ジョブ達成（質）**
- 参加者の満足度
- 対話の深さ
- 安心できる雰囲気

**30%：コミュニティ（関係性）**
- リピート率
- 繋がりの質
- NPS

**10%：規模（量）**
- 参加者数
- 開催頻度
- 認知度
