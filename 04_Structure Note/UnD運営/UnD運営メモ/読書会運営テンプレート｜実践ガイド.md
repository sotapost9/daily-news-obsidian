# 読書会運営テンプレート｜実践ガイド

作成日：2025-10-22

---

## 📋 このテンプレートの使い方

各回の読書会で以下のテンプレートをコピーして使用してください。
改善しながら自分なりの型にブラッシュアップしていきましょう。

---

## テンプレート1：開催前チェックリスト

### 1ヶ月前
- [ ] 次回の日程を決定
- [ ] テーマを決定
- [ ] 関連本リストを作成（3-5冊）
- [ ] 告知文を作成
- [ ] SNS投稿（X、note、Slack）

### 2週間前
- [ ] リマインド投稿（SNS）
- [ ] 参加状況の確認
- [ ] 進行台本の準備
- [ ] Zoom URLの準備

### 1週間前
- [ ] 参加者にリマインドメール送信
- [ ] 「本を読んでなくてもOK」を強調
- [ ] 当日の流れを共有
- [ ] 事前アンケート送付（任意）

### 3日前
- [ ] 最終リマインド
- [ ] 初参加者への個別メッセージ
- [ ] 進行台本の最終確認

### 前日
- [ ] 参加者リストの確認
- [ ] Zoom接続テスト
- [ ] 資料の最終準備
- [ ] 「気楽に参加してね」メッセージ

### 当日（開始前）
- [ ] 30分前：会場（Zoom）準備
- [ ] 15分前：接続確認
- [ ] 10分前：Zoom開放、BGM（任意）
- [ ] 開始直前：深呼吸、笑顔

## テンプレート3：進行台本（60分版）

### 【0-10分】オープニング

**00:00-00:02 開始の挨拶**
```
皆さん、こんにちは！〇〇です。
今日もご参加ありがとうございます。
第X回倫理読書会を始めます。
```

**00:02-00:05 今日のテーマ・流れの説明**
```
今日のテーマは「〇〇〇」です。
[テーマの背景を1-2分で説明]

今日の流れはこんな感じです：
1. 近況共有（全員で）
2. 本の紹介・対話
3. クロージング

ルールを再確認：
・本を読んでなくても大丈夫！
・正解はありません、対話を楽しみましょう
・知識のひけらかしではなく、共感と利他で
・全員が話せる場を大切に
```

**00:05-00:10 近況共有（アイスブレイク）**
```
では、近況共有から始めましょう。
お一人1分以内で：
・お名前（呼んでほしい名前）
・最近どうですか？何か気になってることは？
・今日話したいことがあれば

では、〇〇さんからお願いします！
```

---

### 【10-50分】メインセッション

**パターンA：テーマ対話型**

**10:00-10:15 テーマ導入**
```
今日のテーマ「〇〇〇」について、
まず私から少し話題提供をさせてください。

[5分程度で話題提供]

皆さんはこのテーマについて、
どんなことを感じますか？
```

**10:15-10:35 小グループ対話（ブレイクアウト）**
```
では、3-4人のグループに分かれて、
このテーマについて対話してみましょう。

問い：
- 〇〇について考えたことは？
- 最近〇〇だと感じた経験は？
- 〇〇と聞いて思い浮かぶ本や出来事は？

20分後に戻ってきてください！
```

**10:35-10:50 全体共有**
```
お帰りなさい！
各グループでどんな話が出ましたか？
印象的だった話を共有してください。
```

---

**パターンB：本紹介型**

**10:00-10:40 本の紹介タイム**
```
では、皆さんが読んだ本や
気になっている本を紹介してください。

お一人3-5分で：
- どんな本か（タイトル、著者）
- なぜその本を選んだか
- どこが推しか、印象的だったか
- 今日のテーマとの関連

質問や対話も歓迎です！
では、〇〇さんからお願いします。
```

**10:40-10:50 テーマの共通性を探る**
```
色々な本が出てきましたが、
共通するテーマや視点はありましたか？

今日のテーマ「〇〇〇」と繋がる部分を
一緒に探してみましょう。
```

---

**パターンC：フリー対話型**

**10:00-10:05 問いの投げかけ**
```
今日は、こんな問いを投げかけてみます：

「〇〇〇〇〇〇？」

まず、自分の経験や考えを
少し思い浮かべてみてください（1分）
```

**10:05-10:45 フリー対話**
```
では、自由に対話していきましょう。
話したい人から話してください。

私はファシリテーターとして、
交通整理をしますね。

全員が話せるように、
話が偏らないように気をつけます。
```

**10:45-10:50 対話の振り返り**
```
今日の対話を振り返って、
どんなことが見えてきましたか？
```

---

### 【50-60分】クロージング

**50:00-50:55 一言ずつ**
```
そろそろ時間ですので、クロージングです。

お一人30秒ずつ、
今日の気づきや感想を一言お願いします。

では、〇〇さんから！
```

**50:55-51:00 事務連絡・感謝**
```
ありがとうございました！

【事務連絡】
- アンケートを後ほど送ります（2分で終わります）
- 次回は〇月〇日、テーマは「〇〇〇」を予定
- 参加者限定Slackもあります（任意）
- 今日の様子はnoteで共有予定です

今日も素敵な時間をありがとうございました。
また次回お会いしましょう！
```

**51:00- 延長戦（任意）**
```
このまま残って雑談したい方は
どうぞ残ってください！

時間のある方はぜひ。
（10-15分程度）
```

   - [ ] 読みたい本が見つかった

## テンプレート10：緊急時対応

### トラブルシューティング

**参加者が来ない場合**
- 5分待つ
- 来た人だけで開始（3人以上いれば）
- 少人数ならより深い対話に

**技術トラブル（Zoom等）**
- 予備のZoom URLを用意
- Slackで代替連絡手段
- 最悪は延期判断を早めに

**対話が盛り上がらない場合**
- ブレイクアウトで小グループに
- 自分から具体例を出す
- シンプルな問いに戻る

**一人が話しすぎる場合**
- 「他の方の意見も聞いてみましょう」
- タイムキーパーを明示
- ブレイクアウトで分散

**批判的な参加者がいる場合**
- 「色々な意見がありますね」と受け止める
- 「では別の視点から...」と話題を変える
- 休憩を挟む
- 終了後に個別フォロー

## 最後に

このテンプレートは、回を重ねるごとに
あなた自身の型にカスタマイズしてください。

大切なのは：
- 参加者が安心して話せる場を作ること
- 利他と対話を大切にすること
- 楽しみながら続けること

型があるからこそ、遊びを取り入れられます。
まずは型を作り、そして自分らしさを加えていきましょう。
