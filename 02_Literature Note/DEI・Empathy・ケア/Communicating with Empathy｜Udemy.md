# Types of Empathy

## Cognitive

1. 状況についての詳細や全体像を時間をかけて掴む（最初にCognitiveがくるべき）
2. 他者の理解を深める
3. ジャッジなしで、他者の話を引き出す
4. 「認知的エンパシーは、少し厳しく攻撃的に見えることがある」
    1. 感情を共有するわけではなく、あくまで頭で理解するタイプの共感。
    2. 時に冷静すぎたり、論理的に相手の気持ちを見抜いたりするので、冷たい、鋭いなどという印象を与えることがある
    3. 相手からすると「気持ちをわかってくれてる」というより「分析されている」と感じることがある
5. しつこかったり冷たくならないための対策
    1. 相手のボディーランゲージを見る
    2. トーンを感じ取る
    3. 多すぎるな、と思われる質問の量を自覚する
6. Emotional
7. Compassionate

## Emotional

- その人の経験したシュチュエーションを感じ取ること
- 注意
    - これだけだと原因を誤解したまま共感することになる。
    - EmotionalとCognitive両方大事

## Compassionate

- 定義
    - サポートや援助を実際にすること
- 注意
    - 認知、感情をすっ飛ばして援助をすると、的外れになることが多い。
    

# Empathic Listening

- 定義：目の前の人に集中すること。言葉や内容やトーンだけを聞くのではなく、ボディーランゲージも含めて聞く。
- 難しさ
    - 修行が必要。すぐに習得できなくてもOK。
    - 脳、体力、精神のエネルギーをかなり使う。
- 重要性：
    - ちゃんと聞いてくれる感じが得られる。大事な時計の修理で、言葉だけを聞かれるか、不安まで読み取ってくれるかで安心感が全然違う
    - ちゃんと聞いてくれてる、と思ってもらうだけで信頼につながる。
- ３A
    - Attitude
        - こころを開いて聞く。成長のマインドセット。
        - 素直さ？
        - まずはポジティブに聞き入れること。深い議論はあとで行う。
    - Attention
        - 全身全霊を持って聞き込むこと
        - 他の考えや、主観など、聞くことの妨げを一旦おいておくこと。
    - Adjustment
        - 自分の態度、思想、注意を柔軟に変更すること
- Empathy MapもEmpathyの1つ。
    - 客観的に多角的に理解しようとする。かつEmotion（Felt)も扱っている。CognitiveとEmotional両輪で理解しようとしている。
    - ここに書いてないCompassionalがリサーチ結果の行動ってことになるのか。すげー。
