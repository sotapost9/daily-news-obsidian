# **3. 各部・各章の内容と論点**

**第1部　「情弱／情強」という図式を書き換える**

1. “情弱”とは誰か？
    - ネットスラングとしての「情弱」は「最新情報を追えない怠惰な人」への嘲笑語だが、本書はこれを「情報強迫性障害」（author の独自概念）――常時接続をやめられず情報に追われる状態――まで含め再定義。
    - “情弱”は固定的属性ではなく、誰もが置かれ得る流動的ポジションであるとし、生政治（フーコー）の視点で「知の欠乏」ではなく「データ化される生の脆弱さ」に光を当てる。
2. 
3. 社会的マイノリティとデジタル・ディバイド
    - 障害者、高齢者、生活困窮者が経験する〈アクセスの壁〉と、健常者が自ら招く〈情報過多ストレス〉を対比。
    - 「デジタル包摂政策が“万能の解”となるわけではない」という警句とともに、プラットフォーム依存が可視化する新しい不平等を整理する。
4. 

**第2部　ビッグデータをめぐる三つの神話**

1. “やさしさ”の神話
    - ウェアラブル・ライフログや特定健診による「予防介入」は一見「優しい介護」のようで、実際には行動規範の内面化（“自粛される生”）を促す。
    - 介護予防アプリの UX を事例に、誘導デザインが自由意思をどう狭めるかを検証。
2. 
3. “愛”の神話
    - 行政番号（マイナンバー）や介護保険が掲げる「見守り」「つながり」というレトリックを問い直す。
    - ビッグデータは「私を理解してくれる親密なまなざし」の体裁を取りながら、福祉給付の線引きを自動化し、排除の根拠にもなる――これを著者は“擬制される生”と呼ぶ。
4. 
5. “真実”の神話
    - AI 与信や市民スコアリング（Citizen‑Rated Society）を例に、予測アルゴリズムが「真実」を名乗る過程を解剖。
    - 「正確さ」を盾にしたアルゴリズムの偏り・差別リスクを指摘し、スコア管理社会が生む “適正化される生” を批判する。
6. 

**第3部　「生の技法」としてのコンヴィヴィアル・メディア・リテラシー**

1. 障害当事者の実践から学ぶ
    - イリイチの〈コンヴィヴィアリティ〉を手掛かりに、「使いこなすことで社会関係をひらく」メディア・リテラシーを提案。
    - 障害者がスイッチインターフェースをハックし自分の生活リズムを取り戻す事例を挙げ、「ツールとの対話的関係」が万人にとっての〈生の技法〉になり得ると論じる。
2. 
3. 終章　“情報弱者”となる私たちへ
    - 高度情報化のなかで「いつでも誰でも情弱になる」現実を直視し、①データ過信から距離を取る間主観的判断、②選択的無知の肯定、③小規模コミュニティ間の〈翻訳〉 を提案。
    - 最後に「情報に負けない身体感覚の再獲得」を呼びかけて締めくくる。
4.

| **キーワード** | **概念の要点** | **関連理論** |

# **まとめ**

本書は「情報弱者＝リテラシー不足」という常識を覆し、“データで規格化される私たち”こそが脆弱であるというパラダイム転換を提示します。

「情報を“持つ／持たない”の次元」から「情報とどう付き合うかという身体的・倫理的次元」へ議論をスライドさせ、障害当事者が実践してきた〈ツールとの対話〉を社会全体の生き方へ拡張したことが最大の貢献点です。

インクルーシブデザインやDEIに取り組む読者にとっては、技術を “優しく”するだけでなく、“主導権を返すこと” が本質だと気づかせてくれる一冊と言えるでしょう。
