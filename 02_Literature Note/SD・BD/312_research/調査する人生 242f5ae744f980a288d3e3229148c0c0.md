## **学びの全体像 ── 6 つの視点で再編成**

### **1. 質的調査の**

### **存在意義と射程**

- **他者理解の営み**
    - 「調査＝データ集め」ではなく、相手固有の合理性を身体ごと引き受けて理解する作業。
    - 例外や揺らぎを掘り出し、「一概に言えない」を可視化することで定説をぼかす。
    - 既存のロジックを増やすのではなく、**問いと仮説を増殖**させる研究姿勢。
- **わかり合えない論への抵抗**
    - 「他者はわからない」で思考停止しない。
    - 異国・被害・エクストリームな行為でも、**共有できる合理性は意外と残る**。
    - それでも残る「わからなさ」を認めつつ、テンプレを越えた理解を目指す。
- **質的調査＝他者理解**という原点を常に再確認する。

### **2.**

### **倫理とコミットメント**

### **―― “年単位”で沈む覚悟**

- **長期関与こそ倫理**
    - 年単位で関わること自体が責任を担保する。
    - *社内 ID の調査*でも、単発ヒアリングで終わらせず継続的に伴走する設計を。
- **複雑な対人関係**
    - 尊敬・敬意・尊重が出発点。「非合理」に見える行為にもその人の合理性がある。
    - インタビュイーにも何かを“返す”──語り合う場そのものを価値に。
- **暴力・加害をどう書くか**
    - 深い理解は「自分もそうしたかも」という情状酌量を呼び込む。
    - 加害者を理解・救済してよいのかという政治的葛藤に答えは出ていない。
- **「寄り添う」が嫌いという感覚**
    - 上間陽子の〈めんどくせえが仕方ねえ〉スタンス——ケアの現場でのリアルな温度感を忘れない。

### **3.**

### **リサーチャーの位置取りと自己制御**

- **“平均値”を内蔵する**
    - 自分の中に客観的なスケールを保持することで、現場との差分が際立つ。
    - 予想外の出来事が起きても「自分のせい」と矮小化せず、環境の特徴として捉える。
- **全身で浸かりつつアイロニーを保つ**
    - どっぷり参与しながらも、どこか一歩引いて違和感を検知する“遊び”と余裕が不可欠。
- **上下どちらから入るかを選ぶ**
    - ヤンキー集団の“パシリ”のように下に入る方が密なコミュニケーションを得られる場合もあれば、上位ポジションからでしか見えない層もある。
- **能力格差の自覚**
    - “境界面”に立てるのは、一定の知的・社会的スキルがあるからこそ。
    - 能力を磨くことが、逆に能力差を乗り越えて対等に向き合う条件になる。

### **4.**

### **聞き取りの作法とスキルセット**

| **スキル** | **ポイント** | **チェックリスト** |
| --- | --- | --- |
| 質問設計 | 質問リストは地図。だが**実践知（即興）**がなければ聞けない | ☑ 「何が起きたか」だけでなく「どう感じたか」を必ず追う |
| 反射神経 | 面白い端緒に**瞬時に食いつく** | ☑ 予定の質問を一旦捨てる勇気 |
| 量的経験 | テーマ別に200人を聞けば勘所が身体化する | ☑ 繰り返しの中で暗黙知を蓄積 |
| 時間投資 | “じわじわ”解釈する——理由を急かさず待つ | ☑ 理由探しを強制しない、沈黙もデータ |
| 相互性 | インタビュー自体を相手の変容機会に | ☑ 「教えてもらう場」ではなく「共に考える場」に設計 |

### **5.**

### **DEI・政治性のレンズ**

- **見えない弱者を浮かび上がらせる**
    - DEI 視点でジェンダーや障害を読むと、裕福でも DV を受ける家族など“計測から漏れる弱者”が見える。
- **脱資本と生活保障の現実**
    - 理想としての〈脱資本〉を語っても、「結局は飯が最優先」という現場感覚を踏まえる。
- **加害者理解の境界**
    - 被害／加害という二項を越えた複雑な合理性と向き合うとき、研究者はどこまで踏み込むか。

### **6.**

### **次のアクションへのヒント（あなたの文脈で）**

1. **社内 ID 調査の設計**
    - 年単位で追えるテーマを設定し、担当者ローテーションではなく“同じ調査者が付き合い続ける”体制を提案。
2. **ベースラインの可視化**
    - 自分の“平均ライン”を言語化したチェックリストを作り、毎回のフィールドノートで「差分メモ」を残す。
3. **スキルトレーニング**
    - 反射神経を鍛えるために、週 1 回はアドリブ形式の 10 分インタビューを実施。
4. **DEI の組み込み**
    - 課題設定段階で「数値化されない弱者」の仮説リストをつくり、聞き取り対象を拡張。
5. **倫理コミットメントの共有**
    - 調査開始時に「最低〇年は継続」「成果は相互還元する」などの覚書をチームで作成。

## **各章の詳細**

### **序章**

- 岸は「調査＝データ収集」ではなく「他者の合理性を理解する営み」であり、そのために研究者が〈時間・身体・感情〉をそっくり投じるのだと宣言。
- データの「理論化」は常に暴力性を帯びる可能性があることを自覚したうえで、「どう跳躍するか」が本書の通底テーマ。

### **第１回　打越正行（沖縄の暴走族・ヤンキー研究）**

| **キー概念** | **内容** |
| --- | --- |
| **パシリとしての参与観察** | 研究者自身が「荷物持ち」等を引き受け、ネットワークに「埋め込まれる」ことで信頼を得た。 |
| **暴力を書く責任** | 加害者（恋人を殴る男など）にも敬意を抱かざるを得ない複雑さを、どう文章化するか。 |
| **地元／共同体の両義性** | “優しい共同体”には見えない排除も潜む。ヤンキー文化の急速な衰退と就労環境の悪化を10年越しで記録。 |

### **第２回　齋藤直子（被差別部落の結婚差別）**

- 20年以上にわたり「見えない壁」に当たる当事者から生活史を聞き取る。差別は数量化しにくいが、語りを束ねることで「そこに在る」ことを可視化できる。
- 比較研究を求める外部の声より、当事者が語る「経験の実在性」を優先。

### **第３回　丸山里美（女性ホームレス・食と農の社会学）**

- 人の行為は意志に還元できず、偶発性と環境が絡み合う「微妙なところ」を書きたいと語る。
- 語りの矛盾や沈黙も含め丁寧に掘り下げ、「単線的な物語」に回収しない書き方を模索。

### **第４回　石岡丈昇（マニラのボクサー／肉体労働者の身体文化論）**

- **身体と時間のエスノグラフィー**：走り、働き、殴り合う「からだ一つ」の世界に研究者自身も飛び込む。
- 「生きることを正面に据えると威勢よく言えない」——被調査者の逞しさと脆さを同時に記述する姿勢が語られる。

### **第５回　上間陽子（沖縄の少女支援と調査）**

- 元は「強いコギャル」の文化を描きたかったが、性搾取と暴力の現場に向き合いシェルター〈おにわ〉を開設。
- 研究と支援が絡み合うとき、距離の取り方ではなく「最後までそこに居続ける責任」が問われる。

### **第６回　朴沙羅（在日コリアン家族史・オーラルヒストリー）**

- *「人生を書くことはできるのか」**が核心テーマ。１〜２時間のインタビューで90年の人生をどう扱うか。
- 語りは「裏が取れない私的経験」に満ちるが、歴史的事実との交差点で読むことで個人史を公共化できると示唆。

## **横断的な論点**

| **論点** | **概要** |
| --- | --- |
| **長期コミットメント** | ６人全員が「年単位」を当然視。時間を差し出すこと自体が倫理的担保となる。 |
| **解釈＝暴力問題** | データを「抽象化」する跳躍は不可避。その暴力性を自覚しつつ、「書くしかない」と前に進む覚悟を共有。 |
| **研究と実践のあわい** | 上間（支援）、打越（地域活動）、朴（家族史の承継）など、調査が社会的介入と重なる場面を包み隠さず議論。 |
| **他者の合理性理解** | 「わかる」ことは同情でも統計的因果でもなく、相手がその場で示すロジックを自分の身に引き受けること。 |
