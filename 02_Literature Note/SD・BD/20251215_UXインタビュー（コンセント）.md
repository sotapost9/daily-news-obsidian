- 2. 何をどのように引き出すか
	- 基本：１つのアクションに対して、5つの観点から聞いていく
	- **ポイント**
		- **意見ではなく、エピソードを聞く**
		- **相手に興味を持つことが大事**
	- ![[Pasted image 20251215172945.png]]
	- **①アクションに対して、具体的にどんな行動か？**
	- **②具体的に、どんな状況か？**
	- **③どんな心理状況か？**
		- ワクワクしてたのか、不安だったのかなど、メンタルをその人の言葉で聞く
	- **④どんな経緯・背景？**
		- その状況や行動に至るまで、どのようなプロセスだったのか
		- 深堀のしがいがある
	- **⑤仮に違う状況なら？**
		- 仮に違う状況でも、そうなっていたか
		- 聞いてみることで、その人の価値観が見えてくる。

- 振り返り
	- 発見
		- インタビュー中に​気が​ついた​ことは、​深掘りは​すべきだけど、​自分の​中で​言語化は​仕切らない​ほうが​いいと​感じた。​直感的に​「これは​いいこと​聞けた​/ふかぼろう」とだけ認識できれば​いい​
		- オープンクエスチョンに固執しないほうがむしろいいかなと実践を通じて感じました。クローズドクエスチョンをちょくちょく挟むほうがスムーズに行きそう。しかしやはり極力オープンクエスチョンにしようという意識は大事。
		- 質問の逃げを用意しておくことは大事。盛り上がらなかったときに、この角度から聞き直せばいいという手札をたくさん作っておく。今回なら、買い物課題という目的に対し、「最近」「実店舗/EC」「高い」「後悔」など、関連しそうな問い口を複数用意しておいたのが安心感とスムーズさにつながった。
	- 実践
		- 意外といけると思った
		- 質問は​決めすぎない​ほうが​いいと​思った。​むしろ​聞かなくてはいけない​質問が​たくさん​あると、​覚える​ために​リソースが​必要と​なり、​大変そうだと​思った。​
