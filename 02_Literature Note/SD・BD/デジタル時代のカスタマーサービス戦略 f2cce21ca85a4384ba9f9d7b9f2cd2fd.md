# 序章

- 「カスタマーサービスは、もはや顧客からの問い合わせに対応する企業の一部門ではない。マーケ＆セールス、デザイン＆プロダクション、財務＆アドミと並び、**企業を支える4本目**のとして位置付けるべきである」
    - サービスデザイン時代だからこそ、顧客と企業の大きなタッチポイントを快適で愛のあるものにするというのは重要な流れなんだな。
- 「カスタマーサービスを戦略的に捉え直すということは、競合他社には簡単に真似できない真の競争力としてサービスを再定義することであり、人材、テクノロジー、リーダーシップのあり方も同時に再定義する必要がある。」
    - 簡単に真似できないというのがポイント。カスタマーサービスの再定義は大変だからこそ、圧倒的な競争力差別化につながる。
- 「デジタル化が進んだ結果、顧客は困りごとを助けてもらう際に、**新しい生活様式として定着するテクノロジーもあれば、親切に寄り添ってくれるヒューマンなサポートを渇望するかもしれない。」**
    - 生活に溶け込むテクノロジー、また逆に親切に寄り添う温かみ。あるいはその両方？→Orihime、おせっかいばあば
- 「一見矛盾した人とテクノロジーの関係の背後には、人々のコミュニケーション手段の劇的な変化がある。私たちは、**ヒューマンな感情を伝える際にも絵文字などデジタルに頼っている。**」
    - 感情を伝えるためのデジタル。Orihimeもホスピタルを提供したいマイノリティのための架け橋となっている。形式からの選択の楽さ、顔を映さずにすむ心理的安全性、場所を選ばないことなど、それらによってできることは多い。
- 「効果的なディライトの48％がデジタルチャネルを通じて行われたものだった。つまりカスタマーサービスは、人かデジタルかという問題ではなく、その二者の融合によって、CXは能動的になり、予知的になり、寄り添ったものになるだろう。」
    - 現代はヒューマンによるものだけじゃないんだ。人とデジタルチャネル両方のカスタマーサービスを考えなけれな行けない。

# 第１章：カスタマーサービスを戦略的に捉える

- 「ほとんどの顧客は苦情を申し立てないだけでなく、企業に悪いインパクトを与える」
    - 「低価格の消費財への苦情申立ては不満がある顧客のうち５−１５％にとどまる。また苦情の申し出先は購入店舗になってしまうので、メーカー本社には声は届きづらい。」
    - 「高額商品でトラブルが深刻だと、苦情の申し立ては２０−50％に上昇する。しかしマネージャや本社に届く段階でその数は10分の1にまで減ってしまう」
- 苦情が出ない理由
    - ①苦情を申し出ても何も変わらないと考えている
    - ②企業との関係が拗れることを嫌がる
    - ③トラブルにはむかつくのだが、苦情を申し立てるほどではない
    - ④どこに苦情を申し出て良いのかわからない
- スタッフに権限を与える
    - 顧客はサイトで解決できないことを初めて問い合わせる。そこでほとんどの問題は解決できなければいけない。
    - そのためにはスタッフに十分な知識を訓練することだけでなく、権限を与えることが必要。
        - 局戻しを整理していたのはこれが理由か
- コールスタッフが悪いわけではない
    - いまだにクレームに対してスタッフを叱りつけることがあるらしいが、問題のほとんどはシステム。
- 営業よりもサービス担当者？
    - ほとんどの顧客は営業担当よりもサービス担当者からのアドバイスを素直に受け入れる
        - サービス担当者にも営業できるツールを用意しておくことが必要なんだ
