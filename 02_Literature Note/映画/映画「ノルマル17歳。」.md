# まとめ
- テーマ
  - 「普通」という名の呪縛：ADHDと社会の「普通のものさし」
- 結論
  - ADHDの人々は「普通のものさし」で測られ問題児と見なされるが、その苦しみは障害に甘えているのではなく、人生スパンで続く本質的な困難である

## 構造
- 矛盾
  - A：「普通」の基準・社会の期待・健常者の視点
  - B：ADHD当事者の日常・理解されない苦悩・生きづらさ
- 示唆
  - 「普通はそんな大事なことを忘れるわけがない」という社会の前提が、ADHD当事者を追い詰める。多様性を認める社会への転換が必要

### 人物分析
- 中心キャラ：ADHD女子高生2人
  - 背景：
    - 財布や鍵をなくしたり、大事な予定を忘れたり、不用意な発言をしてしまう日常
    - 「普通のものさし」で見る人々から「問題児」として見られる
  - 行動と結果（感情・変化・内的評価など）：
    - ADHDをとりまく問題や苦労が人生スパン全体で描かれている
    - 「普通」の呪縛に苦しめられ続ける
- その他キャラ：父親
  - 背景：
    - ADHDを理解しようとしない典型的な大人
  - 行動と結果（感情・変化・メインキャラへの評価など）：
    - 「ADHDに甘えているだけだろ」というセリフが印象的
    - 当事者の苦しみを軽視する社会の象徴
- 自分
  - メインキャラに感じたこと：
    - ADHDの困難が「甘え」ではなく、実際の障害であることを改めて認識した
    - 若干美化されすぎている気もするが、社会課題を自分事化する良い作品

### 社会背景
- 「普通」の基準が硬直的な日本社会。発達障害への理解不足と、「努力すればできる」という精神論が根強い
