# まとめ
- テーマ
  - 血縁よりも絆：真の「父」とは何か
- 結論
  - 父になるとは、自分のペースを押し付けず、相手の弱さと自分の弱さを受け入れること

## 構造
- 矛盾
  - A：完璧主義・優秀さ・支配
  - B：不完全性・弱さ・受容
- 示唆
  - 完璧であろうとする姿勢は孤独を生む。真の父性は、自分と子供の弱さを受け入れた時に生まれる（A vs Bの二項対立から導き出されるもの）

### 人物分析
- 中心キャラ：父（主人公）
  - 背景：
    - 優秀だが共感力が低い
    - 仕事での職位や地位を築き上げてきた
    - 完璧主義者で、決めたルールを押し付ける傾向
  - 行動と結果（感情・変化・内的評価など）：
    - 【父になれない苦悩】妻からも本当の息子からも距離を取られ、孤独になっていく
    - 【転換点】子供の家出をきっかけに「すぐにお父さんって呼ばなくてもいいからな」と自分のペースを押し付けることをやめた
    - 【変化】「俺も家出したんだ。母親に会いたくて」と自分の弱さを認め、子供と向き合うようになった
    - 子供の遊びに付き合い、子供のレベルに合わせるようになり、子供も心を開いた
- その他キャラ：育ての父
  - 背景：
    - 本当の息子ではないが、子供から慕われている（中心キャラとの対比）
  - 行動と結果（感情・変化・メインキャラへの評価など）：
    - 子供が家出してまで会いに行くほど慕われている
    - 主人公にとって「父」のあり方を示す鏡像的存在
- 自分
  - メインキャラに感じたこと：
    - 頭ごなしに怒鳴り、理不尽を押し付ける姿は、自分自身の過去や上司の姿と重なる
    - 弱さを認めた瞬間から人との関係が変わる様子に、真の成長を感じた

### 社会背景
- 現代社会における完璧主義と競争社会の弊害。仕事での成功が家庭での幸福を保証しないというテーマは普遍的

- 父になれない苦悩
	- 優秀だが共感力の低い父が、妻からも本当の息子からも距離を取られてしまう。誰からも味方になってくれない辛さがじわじわとk感じられてくる
	- 「誰かに愛されたいのか？お前らしくないけど」というセリフが父の弱さを的確に射抜いてくる
	- 完璧になろうとするが、不器用な父はどんどん孤独になっていく。気付きあげた仕事での職位や地位もあっけなく無駄になり、役にも立たない。
	- 頭ごなしに怒鳴る。決めたルールの背景を説明せず「そういうもんだ」と押し付ける。上司からの理不尽と全く同じことを子供にしている
- 父への転換
	- 子供が家出してまで育ての父に会いに行ってしまったことをきっかけに、「すぐにお父さんって呼ばなくてもいいからな」と、自分のペースを押し付けることをやめた。この瞬間に、父になったということか
	- 「俺も家出したんだ。母親に会いたくて」と呟く。自分と、子供の弱さを自然と受け入れた瞬間。
	- すると子供も「お父さん」と呼び始めるようになった。
	- 子供の遊びに付き合う。やられる、演じる。子供のレベルに合わせる、というか世界観に付き合う。
	- 一人の人間として子供に、自分よりも弱い人間に向き合う。子供もそれを敏感に察知し、こころを開く
