# まとめ
- テーマ
  - ルールに縛られた人間に、ラポールは見つけられない
- 結論
  -人と人の本質的な繋がり（ラポール）は、ルールの内側では生まれない。逸脱の中でこそ、互いの真実が露わになり、信頼が立ち上がる

## 構造
- 矛盾
  - A：ルール・正しさ・秩序・社会の常識
  - B：逸脱・真実・切実な欲望・本質的なつながり
- 示唆
  - 社会のルールや正しさの中では、誰かを本当に信じることも裏切ることもできない。ラポールは逸脱の中でしか生まれない

### 人物分析
- 中心キャラ：娘・楓と父・原田智
  - 背景：
    - 父が失踪したことをきっかけに、娘が真実を探しに行く
    - 父は自殺志願者を実際に殺す男と出会い、共犯関係になる
  - 行動と結果（感情・変化・内的評価など）：
    - 【父】「死にたい」と願う相手の声を本気で受け止め、ルールの外に出てしまった相手に自らも逸脱して近づく
    - 【楓】学校の先生やシスターの「正しい対応」は孤独に届かない
    - 【ラスト】父を裏切って警察を呼びながらも、ボールのない卓球を続けることで「ラリーを続けよう」とする意思だけが残る
- その他キャラ：ムクドリ（車椅子の女性）
  - 背景：
    - 他の自殺志願者とは異なり、明確に「死にたい」と望む
  - 行動と結果（感情・変化・メインキャラへの評価など）：
    - 殺される直前に「雑魚が」と吐き捨てる
    - 本当に死を選べる強さを持つ者の意志の重さを示す
- 自分
  - メインキャラに感じたこと：
    - ルールの中では誰かの痛みを丸ごと受け止めることはできない
    - ボールのない卓球＝形式が消えた後に初めて立ち上がる深い対話の美しさ

### 社会背景
- 正しさと秩序を重視する社会では、ルールの外にいる人々は孤立する

**ルールに縛られた人間に、ラポールは見つけられない**

私たちは、日々「ルール」の中で生きている。  
法律、倫理、マナー、役割、通念、空気——それらは社会の秩序を維持するために存在しているが、同時に、私たちの関係性にも無意識の「制約」を課している。  
相手の気持ちを察するよりも、正しさを守る。感情をぶつけるよりも、場を乱さないことを優先する。

けれど私は、映画『さがす』を観て、改めて思った。  
**人と人の本質的な繋がり、ラポールは、ルールの内側では生まれない。逸脱の中でこそ、互いの真実が露わになり、信頼が立ち上がるのでないか。**

## 死を願う者にしか見せない顔

『さがす』は、父・原田智が失踪したことをきっかけに、娘・楓が真実を探しに行く物語だ。しかし、これは単なるサスペンスや親子の絆を描いた作品ではない。  
その奥には、**「死にたい」と願う人間と、それを“救済”しようとする人間の、異様なラポールの物語**が隠されている。

智は、ある男と出会う。男は、ネット上で自殺志願者を探し、実際にその命を奪っていた。  
これは明らかに法と倫理に反する行為だ。だが彼にとっては、**「死にたい」と願う相手の声を本気で受け止め、行動に移すこと**こそが、救済であり、コミュニケーションだった。

彼が求めていたのは、「制度の中でできる支援」ではなかった。  
**社会が言う“正しいやり方”をすべて無視してでも、相手の内側に手を伸ばすこと。ルールの外に出てしまった相手に、自らも逸脱して近づくこと。**  
そこにしか、彼なりの“信頼”が生まれなかった。

## 「雑魚が」と言い放つムクドリの異質な意志

その中でも異様な存在感を放っていたのが、車椅子の女性・ムクドリだった。  
彼女は、他の自殺志願者とは明らかに違っていた。誰よりも強く、明確に「死にたい」と望み、男に殺してくれと頼む。

殺される直前、彼女は男に「雑魚が」と吐き捨てる。その言葉は、**まさに“本当に死を選べる強さ”を持つ者の、意志の重さ**を象徴していた。  
そしてその強さに、むしろ男の方がたじろいでしまう。自殺志願者を“救済”してきたはずの彼が、はじめて“受け入れきれない死”に出会った瞬間だった。

ここでも、**ラポールは常識の外側にある。生きる・死ぬという最大の境界を超えようとする、その切実な欲望の交差点にしか、本当の繋がりは立ち上がらない。**

## 卓球という“逸脱”の象徴

映画のモチーフとして象徴的なのが、卓球だ。  
父と娘はたびたび卓球を通じてやりとりする。そしてラストシーン——父が罪を犯したことを知った楓は、彼を見つめながら、**ボールのない卓球**を静かに続ける。

ゲームとしての意味は崩壊している。勝敗も、ルールも、進行もない。  
けれど、**そこにだけ残ったのは、“ラリーを続けよう”とする意思そのもの**だった。

これは、形式が消えた後に初めて立ち上がる、深い対話だと思う。  
まるで、インディーゲーム『The Last Game』のように。死を前にした存在が、ボールもなく、得点もなく、ただ“ラリーを続ける”ことで、沈黙の中の愛情を表現しようとする。  
**ルールが壊れたからこそ、心だけが残る。その瞬間にこそ、ラポールは生まれる。**

## ルールの中の対話は、すれ違う

一方で、“ルールの内側”にある関係性はことごとく破綻していた。

学校の先生、シスター、警察——誰もが楓に「正しい対応」をしようとするが、それらは全く彼女の孤独には届かない。  
例えばシスターは、「お父さんはもう帰ってきません。現実を受け入れなさい」と言い放つが、それは**彼女の苦しみの背景に耳を傾けることなく、カテゴリーに押し込むだけの言葉**だった。  
楓はその言葉に怒り、唾を吐きかけて立ち去る。

先生の家に一時的に匿われた時も、彼女は先生の子どもに遠慮し、いちごを食べることもできない。  
「正しい人」「やさしい人」の元にいても、**100%甘えられない関係は、どこかに“ルール”の壁が存在している。**

だからこそ、楓が最後に選んだのは、**“父を裏切ること”＝警察を呼ぶこと**でありながら、その行為すらも卓球という“つながり”で包んで見送る、矛盾の抱擁だった。

## 本当のラポールは、逸脱からしか始まらない

『さがす』が描いたのは、単なる死や殺人の話ではない。  
それは、**社会の正しさが届かない領域にいる人間たちが、それでも互いに手を伸ばす物語**だった。

人は、ルールの中では、「正しさ」と「遠慮」のバランスを取り続けてしまう。  
だが、**そのバランスの中では、誰かを本当に“信じる”ことも、“裏切る”こともできない。**

誰かの痛みを丸ごと受け止めようとしたとき、  
誰かの死に“共犯”として向き合おうとしたとき、  
自分の愛を貫くために罪を背負おうとしたとき、  
そして、ルールのない卓球を黙って打ち続けようとしたとき——

**ラポールはいつも、逸脱の中でしか、生まれない。**
