# Zettelkasten

> **体系的なガイドは [[Obsidian完全ガイド：Zettelkastenと知識管理の実践]] を参照**

## ノートの種類

- **Fleeting Note（ひらめきメモ）**：日々のアイデアの断片や観察を書き留めるメモです。ごく短文や走り書きでも構いません。思いつきを素早く記録し、後で見返すための**一次的なメモ**です 。文章の整合性は気にせず、気軽に書けることを重視します。
    
- **Literature Note（文献ノート）**：本や記事、ウェブ情報などから得た知識を、自分の言葉で要約・抜粋するノートです 。読書メモや論文のハイライト整理がこれに当たります。**他の情報源の内容を自分なりに消化して記録**し、後に永久ノートへ発展させる素材とします 。
    
- **Permanent Note（永久ノート）**：Zettelkastenの中核となるノートで、最も重要です 。FleetingやLiteratureから得た**一つのアイデア**を平易な文章でまとめ直したもので、**誰が読んでも理解できるように書かれ**、単一の論点だけを扱います 。すべてのPermanent Noteは少なくとも一つ他のノートと意味のあるリンクで結ばれ、知識のネットワークを構成します 。
    
- **Structure Note（構造化ノート）**：複数のPermanent Noteを**関連づけてまとめるノート**です 。特定のテーマについて、関連する知識をリンクで束ねて**小論文やガイドのように記述**します  。いわば**「知識の地図」**であり、新たな視点でノート群を整理しながらアイデアを生み出す役割を果たします 。
    
- **Index Note（索引ノート）**：Structure Noteをさらに高次で整理した**目次的ノート**です 。主要カテゴリごとに関連するStructure Noteへのリンクを箇条書きにしたもので、Zettelkasten全体を俯瞰するナビゲーションになります 。Index Noteはノート数が増え全貌が見えにくくなって初めて必要性が出てくるものです 。

- ⭐️日付どうつける？
- 週末にメモを吟味し、永久ノートに発展させれるものは新規でノート作成。元メモをリンクに添えておく。
- リンクを貼る時は、貼った根拠を一言添えるべき
- サブフォルダは最小限にすべき
- ファイル構造
	- 00_Permanent-Notes/ （永久ノート保管用）
	- 01_Structure-Notes/ （構造ノート保管用）
	- 02_Literature-Notes/ （文献ノート保管用）
	- 03_Fleeting-Notes/ （ひらめきメモ保管用）
