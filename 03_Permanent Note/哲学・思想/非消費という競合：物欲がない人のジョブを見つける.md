# 非消費という競合：物欲がない人のジョブを見つける方法

「物欲がない人には売れない」——本当にそうでしょうか？ジョブ理論の視点に立つと、「買わない」という選択肢自体が**代替案のひとつ**に過ぎません。そこには必ず「達成したい進歩（ジョブ）」「うまくいかない事情（障害）」「妥協（現行の代替手段）」が存在します。

この記事では、「非消費」という見えない競合をどう攻略するか、デザイナーとして何ができるかを、ジョブ理論のフレームワークを使って解説します。

## 「物欲がない」は本当にジョブがないのか？

結論から言えば、**NOです**。

「物欲がない」という状態は、ジョブがないのではなく、**「所有の負担」に対する感度が高い**状態です。物を持つことによる負担——お金、時間、注意、保管スペース、メンテナンス、そして「自分らしくない」という違和感——が、得られる進歩を上回ってしまっているのです。

### 物欲の大小は「負担対効果」の閾値の違い

重要な視点転換がここにあります。

> **物欲の大小は『ジョブの有無』ではなく、『負担対効果』の閾値の違い。**

つまり、「物を持ちたくない人」にも、進歩の手段としての物は採用されます。ただし条件があります。

1. **負担を極小化**する（手間、コスト、スペース、注意を最小限に）
2. **初回進歩を最短化**する（Time to First Jobを短く）
3. **後悔の出口**を用意する（返品、買取、サブスク解約など）

## 非消費を競合として解剖する

デザインリサーチの第一歩は、「買わない」という行動を深く理解することです。以下のストーリーフォーマットでインタビューを設計しましょう。

### 1. 状況：どんな文脈でそのジョブが立ち上がる？
- いつ、どこで、誰と、どのくらいの頻度で？
- 例：「雨に降られそうな退勤時」「週に1-2回」

### 2. 期待する進歩：何が"前より良い"なのか？
- 機能的、感情的、社会的な側面は？
- 例：「濡れずに、手を塞がず帰宅したい」「翌日洋服が乾いている安心感」

### 3. 現行の代替：何でしのいでいる？
- 借りる、我慢する、他用途転用、サービス利用？
- 例：「駅の共同傘を使う」「コンビニでビニール傘を買う」「濡れて帰る」

### 4. 障害：購入・所有・使用のどこが面倒／怖い？
- 金銭以外の負担は？
- 例：「持ち歩きが重い」「よく失くす」「乾かすのが面倒」「家に5本もある罪悪感」

### 5. スイッチングの力学（Forces of Progress）

新しい解決策が採用されるかどうかは、以下の4つの力のバランスで決まります。

**(Push + Pull) ＞ (Inertia + Anxiety)**

- **Push（現状の不満）**：濡れるのが嫌、コンビニ傘のコストがかさむ
- **Pull（新しい解決策の魅力）**：軽量、コンパクト、おしゃれ
- **Inertia（現状維持の慣性）**：今の方法で困ってない、めんどくさい
- **Anxiety（新しい解決策への不安）**：本当に使うかな？また失くすかも

**デザインへの示唆：** Push/Pullを強め、Inertia/Anxietyを弱める施策を同時に設計する必要があります。

## 「物欲がない」人の4タイプとデザイン戦略

非消費の理由は一様ではありません。タイプごとに異なるデザイン戦略が求められます。

### タイプ1: 無関心型（ジョブ頻度が低い/痛みが弱い）

**特徴：**
- そもそもあまり困っていない
- たまにしか必要にならない
- 例：「年に2-3回しか傘を使わない」

**有効なレバー：**
- **発生時だけ払う**（レンタル、従量課金）
- **持たない解**（シェアリング、置き傘ネットワーク）

**デザイン例：**
- 駅に設置された傘シェアサービス
- 「使った日だけ100円」の従量制

### タイプ2: 高負担回避型（所有コスト過敏）

**特徴：**
- 物を持つこと自体がストレス
- 維持費、保管場所、メンテナンスが負担
- 例：「家がモノであふれるのが嫌」「管理したくない」

**有効なレバー：**
- **ゼロ負担設計**（軽量、壊れにくい、メンテ不要、小サイズ）
- **出口保証**（下取り、買取、返品制度）

**デザイン例：**
- 超軽量折りたたみ傘（80g以下）
- 「1年後に半額で買い取ります」保証
- 自立式で乾かす手間ゼロ

### タイプ3: 価値観一致重視型（反消費・ミニマル・環境配慮）

**特徴：**
- 「過剰消費」への反発
- エシカル、サステナブルを重視
- 例：「安物買いはしたくない」「環境に配慮したい」

**有効なレバー：**
- **長期耐用**（10年使える品質）
- **修理容易性**（パーツ交換可能）
- **共有/再流通設計**（譲りやすい、売りやすい）
- 誇張しない**静かな語り**（押し付けがましくないメッセージ）

**デザイン例：**
- 生涯保証付き高品質傘
- 骨組みだけ交換可能なモジュール設計
- 「あなたの一本が、誰かの一本に」再流通プログラム

### タイプ4: 選択疲れ/不安型（情報過多・後悔回避）

**特徴：**
- 選択肢が多すぎて決められない
- 「失敗したくない」という不安が強い
- 例：「どれを選べばいいかわからない」「買って後悔するのが怖い」

**有効なレバー：**
- **1択化**（推奨の一本化、「これだけでいい」という安心）
- **お試し→自動継続**（リスクなしで試せる）
- **後悔保険**（30日間使ってダメなら返金）

**デザイン例：**
- 「あなたに最適な一本を診断」パーソナライゼーション
- 「30日間使ってみて、合わなければ全額返金」
- サブスクで最初の1ヶ月無料

## 実践：「負担予算（Burden Budget）」という設計思想

物欲がない人にデザインする際、最も重要な指標が**負担予算**です。これは、顧客が許容できる負担の総量を示します。

### 負担の5つの次元

1. **金銭的負担**：購入費用、維持費、処分費
2. **時間的負担**：購入の手間、使用の手間、メンテの手間
3. **認知的負担**：選択、管理、心配事
4. **物理的負担**：保管スペース、持ち運び
5. **心理的負担**：罪悪感、アイデンティティとの不一致

**デザイン戦略：** これらの負担を**極小化**し、負担予算内に収めることで、非消費層を顧客に変えられます。

### 具体例：折りたたみ傘の負担最小化設計

**従来の傘：**
- 金銭：3000円
- 時間：乾かす手間10分/回
- 認知：「持ってきたっけ？」という心配
- 物理：300g、鞄のスペース圧迫
- 心理：「また失くすかも」という不安

**負担最小化設計：**
- 金銭：**サブスク月500円**（失くしても追加料金なし）
- 時間：**自立式ケース**（置くだけで乾く）
- 認知：**スマホ通知**（「明日雨です、傘を」）＋**GPS追跡**（失くしても見つかる）
- 物理：**80g**、**ポケットサイズ**
- 心理：**失くしても無料交換**、**環境配慮素材**（罪悪感ゼロ）

## デザインリサーチの質問設計：非消費を掘り下げる

「物欲がない」人へのインタビューでは、以下の質問が有効です。

### 質問例

1. **直近3ヶ月で「買わずにやり過ごした」場面を時系列で教えてください。どこが一番面倒でしたか？**
   → 具体的な状況とペインポイントを特定

2. **それを「うまくやれた日／やれなかった日」の違いは何ですか？**
   → 成功/失敗の条件を明らかに

3. **保管・手入れ・処分を考えた瞬間、何を躊躇しましたか？**
   → 負担の種類を特定

4. **過去に買って後悔した体験はありますか？今なら何があれば後悔しなかったですか？**
   → Anxietyの正体と対処法を探る

5. **今の代替手段に点数を付けるなら？その点数が10点上がる条件は？**
   → 改善の余地を定量化

## まとめ：非消費は市場機会である

「物欲がない」は、市場の終わりではありません。むしろ、**未開拓の巨大な市場機会**です。

破壊的イノベーションの理論では、既存市場よりも「無消費（非消費）」の市場の方がはるかに大きいと指摘されています。そこには、切実なジョブを抱えながら、既存の解決策では負担が大きすぎて採用できない人々がいます。

デザイナーとして私たちができることは：

1. **非消費を競合として認識する**：「買わない」という選択肢を深く理解する
2. **負担予算を設計の中心に置く**：5つの負担次元を可視化し、最小化する
3. **4つのタイプに合わせた戦略**：無関心型、高負担回避型、価値観一致重視型、選択疲れ型
4. **スイッチングの力学を操作する**：Push/Pullを強め、Inertia/Anxietyを弱める

最も重要なのは、「物を売る」のではなく、「進歩を売る」という視点です。そして、その進歩を**最小の負担で実現する**デザインを追求することです。
