# PPTと紙書きのTips：実践的な資料作成とアイデア整理術

資料作りに悩んでいた頃、先輩に言われた言葉がある。

「プレゼンはスピードだ。贅肉を削ぎ落とし、テンポ良く濃密に伝える」

それから数年。無数の資料を作り、失敗し、直し、ようやく自分なりの「型」が見えてきた。このノートは、そんな試行錯誤の中で拾い集めたTipsのまとめだ。

PPT作成のコツから、紙に書くことの威力、アイデアの出し方まで。実践して効果があったものだけを詰め込んだ。

---

## 📊 PPT作成の実践Tips

### 1. 表のデザイン：枠線の色と過不足を整える

表を作るとき、一番大事なのは「何を際立たせるか」を明確にすること。

**よくある失敗：**
- 全部の罫線が同じ太さ・色
- セルの結合がバラバラ
- 空白セルが不規則

**改善のコツ：**
- **枠線の色を使い分ける**：外枠は太く濃く、内側の罫線は細く薄く
- **縦横の表は上を全部埋める**：ヘッダー行は空白を作らない
- **過不足を整える**：不要な罫線は削除、必要なところだけ強調

**実例：**
```
❌ 悪い例
┌────┬────┬────┐
│項目A│    │    │
├────┼────┼────┤
│項目B│100 │200 │
├────┼────┼────┤
│項目C│    │300 │
└────┴────┴────┘

✅ 良い例
━━━━┯━━━━┯━━━━━
項目  │ 値1 │  値2
━━━━┿━━━━┿━━━━━
A     │ 100 │  200
B     │ 150 │  250
C     │ 200 │  300
━━━━┷━━━━┷━━━━━
```

### 2. 変数部分だけを際立たせる

フォーマットが同じスライドが複数枚続く場合、**変数部分だけを色や太字で際立たせる**と、聞き手は違いに気づきやすい。

**例：複数ペルソナを比較するスライド**
- ペルソナ名だけを色で変える
- 他のレイアウトは統一
- 差分が一目でわかる

これはプレゼンのリズムを作る上でも重要。聞き手の認知負荷を下げることができる。

### 3. 画面共有では画面をデカくする

オンラインプレゼンでよくやる失敗。

PowerPointを小さい画面で開いたまま画面共有してしまう。聞き手の画面では、スライドが豆粒になる。

**対処法：**
- **右上のホームボタン → 最大化**
- **右下の四角ボタン → フルスクリーン**

些細だけど、これだけで聞き手の集中力が段違いに変わる。

### 4. プレゼンはスピード

[[プレゼンはスピード｜ゲーム桜井]]

> 「ただ短くするということではない。贅肉を削ぎ落とし、テンポ良く濃密に伝える」

ゲーム桜井さんの言葉だ。プレゼンの上手さは、話の長さではなく、**密度**で決まる。

**密度を上げる方法：**
1. **余計な前置きを削る**：「えー、今日はですね…」を全部カット
2. **一文を短くする**：長い文章は2つに分ける
3. **接続詞を減らす**：「そして」「また」「さらに」を削る
4. **具体例を1つに絞る**：3つ並べるより、1つを深掘り

**実例：**
```
❌ 悪い例（冗長）
「えー、今日はですね、皆さんに新しいプロジェクトの進捗についてお話ししたいと思いま
す。このプロジェクトは、実は3ヶ月前から始まっておりまして、そして、現在は順調に進
んでおります。また、いくつか課題もあるのですが…」

✅ 良い例（濃密）
「プロジェクトの進捗報告です。順調ですが、3つの課題があります」
```

## ✏️ 紙に書くことの威力

デジタル全盛の時代に、なぜ紙なのか。

答えはシンプル。**紙に書くと、思考が整理される**から。

### 1. 「考具」から学ぶ：紙と思考の関係

[[考具 240f5ae744f980769a65d27007ea43f7]]

加藤昌治さんの『考具』では、アイデア発想のための21の道具が紹介されている。その多くが、**紙を使う**ことを前提にしている。

**紙の3つの利点：**

**A. 外部記憶化**
- 脳の容量を空ける
- メモしたら忘れていい
- 思考が軽くなる

**B. 視覚化**
- 頭の中が見える
- 関係性が見える
- パターンが見える

**C. 身体性**
- 手を動かすと脳が動く
- 書く行為自体が思考を促す
- デジタルにはない「感触」

### 2. アイデアスケッチ：ペンで図解しながら発想

考具で紹介されている「アイデアスケッチ」は、**ペンで図解しながら発想を可視化**する技法。

**やり方：**
1. A4の紙を横にする
2. 思いついたことを図で描く
3. 矢印や枠で関係性を示す
4. 言葉と図を混ぜる

**デジタルスケッチとの使い分け：**
- **紙（アナログ）**：発散フェーズ、ざっくりアイデア出し
- **デジタル**：収束フェーズ、整理と共有

私の場合、最初は必ず紙。ある程度形になったら、デジタルに移行する。

### 3. ポストイット：1ネタ1枚で大量貼り出し

これも考具の定番技。

**やり方：**
1. 1つのアイデアを1枚のポストイットに書く
2. 壁に貼る（50枚〜100枚）
3. グルーピングする
4. 関係性を発見する

**ポイント：**
- 判断せずに書く（良い悪いは後で）
- 質より量
- 並べ替えながら考える

ホワイトボードでもいいけど、物理的に動かせるポストイットの方が、思考も動く。

### 4. マンダラート：9×9で連想を爆発させる

マンダラートは、**9×9のフレーム強制で連想を広げる**技法。

```
┌───┬───┬───┐
│ 7  │ 8  │ 9  │
├───┼───┼───┤
│ 4  │中心│ 6  │
├───┼───┼───┤
│ 1  │ 2  │ 3  │
└───┴───┴───┘
```

中心にテーマを書き、周囲8マスに関連するキーワードを書く。さらに、その8つをそれぞれ中心として、また8つずつ書く。

8 × 8 = 64個のアイデアが強制的に生まれる。

**使い所：**
- アイデアが出ない時
- 視点を広げたい時
- 抜け漏れをチェックしたい時

## 🧠 思考整理の実践：Zettelkasten

紙に書く話をしてきたが、最終的には「どう管理するか」が問題になる。

そこで、私が実践しているのが**Zettelkasten（ツェッテルカステン）**というノート管理術だ。

[[ノート管理術：Zettelkastenの仕組みと実際にObsidianでやってみた感想]]

### Zettelkastenの5種類のノート

**1. Fleeting Note（思考用ノート）**
- 思いついたことをすぐメモ
- 紙、スマホ、何でもいい
- 後で整理する前提

**2. Literature Note（文献ノート）**
- 読んだ本や記事を自分の言葉でまとめる
- そのままコピペしない
- 自分の解釈を加える

**3. Permanent Note（永久ノート）**
- Zettelkastenの本体
- いつ誰が読んでもわかる文章
- 他のノートとリンクしている

**4. Structure Note（構造化ノート）**
- 関連するPermanent Noteをまとめる
- 知識の地図
- ネットワーク構造を可視化

**5. Index Note（参照ノート）**
- 全体の目次
- Structure Noteのリンク集

### 私の実践

このノート自体が、Permanent Noteとして書いている。

元ネタは：
- Fleeting Note：紙のメモや日記
- Literature Note：『考具』のまとめ、Zettelkastenの記事

これを統合して、一つのPermanent Noteに仕上げた。

## 💡 考具の21の道具（一覧）

参考までに、『考具』で紹介されている21の道具を列挙しておく。

### A. 情報収集系（1-6）
1. **カラーバス**：色を決めて街に出る
2. **聞き耳**：他人の会話を盗み聞き
3. **ちょいメモ**：忘れる前提で即メモ
4. **七色いんこ**：架空キャラになりきる
5. **フォトリーディング**：本を高速スキャン
6. **臨時新聞記者**：現場に取材に出る

### B. 発想拡大系（7-14）
7. **アイデアスケッチ（アナログ）**：ペンで図解
8. **ポストイット**：1ネタ1枚で大量貼り出し
9. **マンダラート**：9×9で連想爆発
10. **マインドマップ**：放射思考でツリー展開
11. **アイデアスケッチ（デジタル）**：PCで検索しながら整理
12. **連想ゲーム**：○と言えば？を投げ合う
13. **オズボーンのチェックリスト**：9視点で変形
14. **ブレーンストーミング**：判断禁止・質より量

### C. 収束・企画化系（15-19）
15. **5W1Hフォーマット**：誰が何をいつどこでなぜどうやって
16. **タイトル**：15文字で核心を言い切る

## 🎯 実践のための3つの原則

最後に、これまでの経験から学んだ3つの原則。

### 1. 量が質を生む

アイデアも資料も、**最初から完璧を目指さない**。

まず量を出す。紙に殴り書きでいい。ポストイットを100枚貼る。ラフな図を10個描く。

質は、量の中から生まれる。

### 2. 外部記憶を持つ

脳は、記憶装置ではなく、**処理装置**だ。

覚えようとするな。書いて、外に出せ。紙に、ポストイットに、ノートに。

脳を空けると、思考が軽くなる。

### 3. 視点をスイッチする

煮詰まったら、**視点を変える**。

色を変える（カラーバス）。役割を変える（七色いんこ）。質問を変える（問いかけの展開）。

視点をスイッチすると、同じ問題が違って見える。

## 最後に：道具は使うためにある

考具も、ポストイットも、Zettelkastenも、**使わなければただの知識**だ。

このノートを読んで「ふーん」で終わらせるのではなく、明日から何か一つ、試してほしい。

紙とペンを用意して、アイデアスケッチを描いてみる。ポストイットを買ってきて、壁に貼ってみる。マンダラートを1枚書いてみる。

道具は、使って初めて「考具」になる。

あなたの思考を助ける道具になることを願って。
