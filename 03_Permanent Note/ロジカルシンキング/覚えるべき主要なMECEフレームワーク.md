
1. 3C分析 (Customer / Competitor / Company)
事業戦略やマーケティング戦略の立案時に基本となるフレームワークです。 

- **Customer（市場・顧客）**: 市場規模や成長性、顧客のニーズなどを分析します。
- **Competitor（競合）**: 競合他社の強みや弱み、市場シェアなどを分析します。
- **Company（自社）**: 自社の強み（リソース、技術、ブランド力など）や弱みを分析します。  
    これら3つの視点で分析することで、事業環境を漏れなく捉えることができます。 

2. SWOT分析 (Strengths / Weaknesses / Opportunities / Threats)
自社の内部環境と外部環境を整理し、戦略を導き出すためのフレームワークです。 

- **Strengths（強み・内部環境）**: 競合他社に対する自社の優位性。
- **Weaknesses（弱み・内部環境）**: 競合他社に対する自社の劣位性。
- **Opportunities（機会・外部環境）**: 市場や社会の変化など、自社にとって有利な外部要因。
- **Threats（脅威・外部環境）**: 規制の変更や競合の新規参入など、自社にとって不利な外部要因。  
    これら4つの要素を組み合わせることで、取るべき戦略の方向性が見えてきます。 

3. 4P分析 (Product / Price / Place / Promotion)
マーケティング戦略の具体的な実行プラン（マーケティングミックス）を立てる際に使用されます。 tyh

- **Product（製品・サービス）**: どのような製品やサービスを提供するか。
- **Price（価格）**: 適切な価格設定はいくらか。
- **Place（流通・場所）**: どのように顧客に届けるか（販売チャネル、立地など）。
- **Promotion（販売促進）**: 製品の魅力をどのように顧客に伝えるか（広告、営業など）。
