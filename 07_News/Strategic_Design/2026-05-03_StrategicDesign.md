## PEST分析

| 要素 | 分析内容 |
| :--- | :--- |
| **Politics (政治)** | 自律型AIエージェントの普及に伴う、ガバナンスと責任の所在の再定義。国家戦略レベルでのAI活用。 |
| **Economy (経済)** | AIによる自動化がもたらす産業構造の劇的変化。効率化だけでなく、新たな価値提案の戦略設計。 |
| **Society (社会)** | AIが「ツール」から「コラボレーター（協働者）」へ。社会全体の意思決定プロセスの変容。 |
| **Technology (技術)** | 小規模で効率的なLLMの普及。エッジデバイスでのAI実行による、よりパーソナルな戦略的体験。 |
