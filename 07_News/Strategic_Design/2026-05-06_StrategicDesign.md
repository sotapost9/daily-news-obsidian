#ストラテジックデザイン

## UX戦略と組織変革におけるデザインの役割

ストラテジックデザインは、単なる製品やサービスの見た目をデザインするだけでなく、ビジネス戦略と組織の変革に深く関わる領域です。Nielsen Norman Groupの「Field Guide to Explaining UX Strategy」では、UX戦略の概念を組織内で共有し、UXビジョン、目標、OKR、成果といった複雑な概念を誰もが理解できる言葉に翻訳することの重要性が説かれています[1]。これは、デザインが経営層と現場をつなぐ共通言語となり、戦略実行の推進力となることを示唆しています。

また、Goodpatch Blogの記事「SIerでもコンサルでもなく、グッドパッチだからこそできること 大企業の組織変革に伴走するデザインストラテジストの流儀」では、デザインストラテジストが、大企業の組織変革において、論理と感情の両面からユーザーに向き合い、UXデザインの視点からコンサルティングを行う事例が紹介されています[2]。これは、ストラテジックデザインが、企業の課題解決や新たな価値創造において、従来のコンサルティングとは異なるアプローチを提供できる可能性を示しています。

サービスデザイナーやデザインストラテジストは、単にユーザーのニーズを満たすだけでなく、ビジネスの目標達成と社会的な価値創出を両立させるための戦略的な視点を持つことが不可欠です。デザイン思考を組織全体に浸透させ、変革をリードする役割がますます重要になるでしょう。

| | | |
| :--- | :--- | :--- |
| **Politics (政治)** | - 企業統治におけるデザインの重要性が認識され、デザイン経営を推進する政策がストラテジックデザインの普及を後押しする可能性がある。 | - 業界団体や標準化団体が、デザイン戦略に関するガイドラインやフレームワークを策定し、企業のデザイン活用を促進する。 |
| **Economy (経済)** | - 競争が激化する市場において、デザインは企業の差別化要因となり、新たな収益源を生み出す戦略的な投資対象となる。 | - デザインコンサルティング市場の拡大や、デザイン人材の需要増加は、経済全体に新たな価値をもたらす。 |
| **Society (社会)** | - ユーザー中心の価値創造が社会的な要請となり、企業は顧客体験の向上を通じてブランドロイヤルティを構築する必要がある。 | - デザイン思考やサービスデザインの概念が一般化し、多様なステークホルダーとの共創を通じて社会課題を解決する動きが加速する。 |
| **Technology (技術)** | - AIやデータ分析ツールは、市場トレンドの把握やユーザーインサイトの抽出を支援し、よりデータに基づいた戦略的なデザインを可能にする。 | - デジタル技術の進化は、新たなサービスやビジネスモデルの創出を加速させ、ストラテジックデザインの適用範囲を拡大する。 |
