# DEI & Minority Insight: 医療・福祉・DEI領域の動向レポート (2026-02-14)

本レポートでは、医療、福祉、メンタルヘルス、およびDEI（ダイバーシティ・エクイティ＆インクルージョン）領域における最新の動向を、サービスデザイナーの視点から分析します。法規制の変化から支援技術の進化まで、インクルーシブな社会設計に不可欠なインサイトをまとめました。

## 1. 医療・福祉領域のPEST分析

現在の医療・福祉領域を取り巻く環境は、法的な義務化と技術的な支援が交差する重要な転換点にあります。以下の表は、各側面における主要な動向を整理したものです。

| 視点 | 主要な動向とサービスデザインへの影響 |
| :--- | :--- |
| **Political (政治)** | **カスタマーハラスメント（カスハラ）対策指針の策定**：2026年10月の施行に向け、障害者への合理的配慮を前提とした対策が義務付けられます。サービス設計においては、提供者と利用者の双方を守る安全なコミュニケーション設計が求められます。 |
| **Economic (経済)** | **処遇改善と人材確保の加速**：介護職の賃上げ詳細が判明し、特定技能1号の在留延長条件緩和など、外国人材の受け入れ（上限16万人超）が加速しています。多様な背景を持つスタッフが働きやすいインクルーシブな職場環境の設計が急務です。 |
| **Social (社会)** | **孤独・孤立対策のシステム化**：欧州を中心に、スウェーデンの「友情休暇」やドイツの「多世代バディ」プロジェクトなど、個人の問題であった孤独を社会システムで解決する動きが活発化しています。 |
| **Technological (技術)** | **支援技術（Assistive Tech）の民主化**：話した言葉を即座に点字化するラベルプリンターなど、専門知識がなくてもアクセシビリティを担保できるツールの登場により、サービス現場での実装ハードルが下がっています。 |

## 2. インクルーシブデザイン・アクセシビリティ最新事例

サービス設計において、マイノリティや特定のニーズを持つユーザーを「最初から」考慮に入れるインクルーシブデザインの重要性が増しています。

### 未成年者へのリサーチ倫理：Consent（同意）とAssent（賛意）
リサーチの現場では、未成年者を対象とする際の倫理的配慮が再定義されています。親権者からの**Consent（法的同意）**だけでなく、子供本人からの**Assent（参加の意志表明）**を個別に取得することが推奨されています。これは、子供を一人の独立した権利主体として尊重するサービス姿勢の表れであり、信頼構築の第一歩となります。

### 多感覚アート体験：だれでもアーティスト展
大阪で開催された「だれでもアーティスト展」では、触覚、音、視覚、コミュニケーションの違いを「制限」ではなく「新しい表現の可能性」として捉え直しています。例えば、目で見るだけでなく「触れるアート」や、音の振動で感じる作品など、特定の感覚に依存しない設計は、障害の有無を超えてすべてのユーザーに豊かな体験を提供します。

## 3. サービスデザイナーへの示唆

医療・福祉領域における「合理的配慮」は、もはや努力義務ではなく、サービス品質を定義する基盤となっています。デザイナーは、個別の「困りごと」を解決する対症療法的なアプローチから、多様な特性を内包した「標準（ニューノーマル）」を再定義するシステム思考へとシフトする必要があります。
