# 【レポートI：デザインリサーチ】
作成日: 2026-03-25
タグ: #DesignResearch #デザインリサーチ #ユーザー調査

## 1. 概況・エグゼクティブサマリー
本レポートでは、デザインリサーチの最新動向をサービスデザイナーの視点で分析します。今回の重要トピックは、**AIによるリサーチプロセスの自動化と「深掘り」の必要性**、**「共感（Empathy）」から「共創（Collaboration）」へのリサーチ手法のシフト**、そして**生活者の微細な習慣変化を捉える「習慣予報」的アプローチ**です。これらは、リサーチが単なる「情報収集」から、ユーザーとの「関係構築」へと進化していることを示しています。

## 2. PEST分析

| 要素 | 内容・動向 | サービスデザインへの影響 |
| :--- | :--- | :--- |
| **P (Politics)** | プライバシー規制の強化と、データ収集における透明性の要求 | ユーザーからリサーチ協力を得るための「信頼（Trust）」の設計が、リサーチの成否を分ける。 |
| **E (Economy)** | リサーチの効率化（AI活用）と、一方で高まる「質的調査」の価値[1] | AIが定量的な傾向を出す一方で、人間による「なぜ（Why）」の深掘りが差別化要因となる。 |
| **S (Society)** | 『感情コレクト』に見る、ユーザーの微細な心理変化の収集習慣[2] | ユーザーが自ら自分の感情や体験を記録し、共有する「セルフリサーチ」の動き。 |
| **T (Technology)** | 生成AIによるプロンプトを通じた「意図（Intent）」の抽出[3] | ユーザーの言葉の背後にある「真の目的」をAIで解析し、リサーチの精度を高める。 |

## 3. 個別記事分析とサービスデザインの視点

### ■ 2026年のリサーチシフト：共創としてのリサーチ
UX Collectiveは、2026年のリサーチにおいて、ユーザーを「エンドポイント」ではなく「パートナー」として扱う重要性を説いています[1]。
- **概要**: ユーザーを調査対象として観察するだけでなく、共に課題を解決する「共創プロセス」に巻き込む。
- **SD視点**: リサーチの目的を「インサイトの獲得」に留めず、リサーチプロセス自体を「ファン化」や「エンパワメント」の手段として設計することが求められます。
- **URL**: [https://uxdesign.cc/10-ux-design-shifts-you-cant-ignore-in-2026-8f0da1c6741d](https://uxdesign.cc/10-ux-design-shifts-you-cant-ignore-in-2026-8f0da1c6741d)

### ■ 習慣の変化から未来を予測する「ヒット習慣予報」
博報堂のヒット習慣予報は、SNSや購買データの分析を通じて、生活者の「兆し」を捉えるリサーチの好例です[2]。
- **概要**: 既存の「ヒット商品」ではなく、これから来る「習慣」を予測する。
- **SD視点**: 「今、何が起きているか」だけでなく「次にどのような行動変容が起きるか」を、微細な兆し（Weak Signals）から読み解くリサーチスキルが、戦略的なサービスデザインには不可欠です。
- **URL**: [https://www.hakuhodo.co.jp/magazine/series/hit-shukan/](https://www.hakuhodo.co.jp/magazine/series/hit-shukan/)

### ■ AI時代のリサーチ：操作から「意図」の解析へ
Nielsen Norman Groupは、AIがもたらすリサーチの変革について、ユーザーの「意図（Intent）」を正確に捉えることの重要性を強調しています[3]。
- **概要**: AIが生成する多様なアウトプットの中から、ユーザーの真の目的に適うものを選別するための評価基準。
- **SD視点**: リサーチャーの役割は、質問を投げかけることから、AIが生成したプロトタイプに対するユーザーの「反応の背後にある文脈」を解釈することへとシフトします。
