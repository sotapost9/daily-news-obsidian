# 【レポートA：DEI & Minority Insight】（2026-04-19）

## 1. 概況とサービスデザイナーの視点
本日のDEIおよびマイノリティに関するニュースでは、**インフラの不足と制度の形骸化**という、サービス提供の根幹に関わる課題が浮き彫りになっています。特別支援学校における教室不足や、障害者就労支援事業所の基準未達は、設計されたサービスが実態としての需要や品質を維持できていないことを示唆しています。サービスデザイナーとしては、単に「支援の枠組み」を作るだけでなく、**持続可能なリソース配分と、現場の運用負荷を考慮したサービスエコシステムの再設計**が求められています。

## 2. 主要ニュース・インサイト

### 教育・就労インフラの危機
| 項目 | 内容 | サービスデザインの視点 |
| :--- | :--- | :--- |
| **特別支援学校の教室不足** | 文科省の調査により、全国で3192教室が不足していることが判明。生徒数の増加に整備が追いついていない。 | 物理的な空間の制約を前提とした、分散型学習やテクノロジー活用の再検討が必要。 |
| **就労継続支援A型の基準未達** | 厚労省の調査で、指定基準を満たさない事業所が19%に上り、常態化している実態が明らかになった。 | 制度の「ハッピーパス」だけでなく、経営難や人材不足という「例外系」を包摂した支援モデルの設計が不可欠。 |

### 社会的包摂と居住の連携
| 項目 | 内容 | サービスデザインの視点 |
| :--- | :--- | :--- |
| **住宅と福祉の連携強化** | 政府が就職氷河期世代を対象に、住宅支援と福祉サービスを組み合わせた3カ年計画を策定。 | 「住む」と「支える」を分断させない、タッチポイントの統合とシームレスな体験設計が鍵。 |
| **盲導犬受け入れの意識** | 盲導犬協会の調査で、受け入れに前向きな回答が6割に達した。 | 心理的な障壁（Will）は低減しているが、物理的な対応（Can）を支える具体的なサービスガイドラインの普及が待たれる。 |

## 3. PEST分析：DEI領域
| 区分 | 分析内容 |
| :--- | :--- |
| **Politics（政治）** | 政府による「住生活基本計画」の決定や、就職氷河期世代への集中支援など、公的扶助の枠組みが具体化している。 |
| **Economy（経済）** | 就労支援事業所の経営悪化が顕在化しており、福祉サービスの経済的持続可能性が問われている。 |
| **Society（社会）** | 特別支援教育への需要増大や、盲導犬等への理解浸透など、マイノリティの権利に対する社会の期待値が高まっている。 |
| **Technology（技術）** | 物理的なリソース不足（教室等）を補完するための、エデュテックやリモート支援技術の実装が急務。 |

## 4. 総括
本日のリサーチから、DEI領域におけるサービスデザインの焦点は「アクセスの確保」から**「質の担保と持続性」**へと移行していることが分かります。特に、制度が現場の疲弊によって形骸化している現状に対し、サービスデザイナーは**「制度と実務のギャップを埋めるサービスプロトタイプ」**を提示する役割を担うべきです。
