# 【レポートA：DEI & Minority Insight】（2026-04-19）

## 1. 概況とサービスデザイナーの視点
本日のDEIおよびマイノリティに関するニュースは、サービス提供の根幹に関わる**インフラの不足と制度の形骸化**という課題を浮き彫りにしています。特別支援学校における教室不足や、障害者就労支援事業所の基準未達は、設計されたサービスが実態としての需要や品質を維持できていない現状を示唆しています [1] [2]。サービスデザイナーとしては、単に「支援の枠組み」を構築するだけでなく、**持続可能なリソース配分と、現場の運用負荷を考慮したサービスエコシステムの再設計**が喫緊の課題として認識されます。

## 2. 主要ニュース・インサイト

### 教育・就労インフラの危機
| 項目 | 内容 | サービスデザインの視点 |
| :--- | :--- | :--- |
| **特別支援学校の教室不足** | 文部科学省の調査により、全国で3192教室が不足していることが判明し、生徒数の増加に整備が追いついていない状況が示されました [1]。 | 物理的な空間の制約を前提とした、分散型学習やテクノロジー活用の再検討を通じて、教育機会の均等化を図る必要があります。 |
| **就労継続支援A型の基準未達** | 厚生労働省の調査では、指定基準を満たさない事業所が19%に上り、その状態が常態化している実態が明らかになりました [2]。 | 制度の「ハッピーパス」だけでなく、経営難や人材不足といった「例外系」を包摂した支援モデルの設計が不可欠であり、現場の実情に即した柔軟な運用が求められます。 |

### 社会的包摂と居住の連携
| 項目 | 内容 | サービスデザインの視点 |
| :--- | :--- | :--- |
| **住宅と福祉の連携強化** | 政府は就職氷河期世代を対象に、住宅支援と福祉サービスを組み合わせた3カ年計画を策定しました [3]。 | 「住む」と「支える」を分断させない、タッチポイントの統合とシームレスな体験設計が鍵となります。これにより、対象者の生活の質を包括的に向上させることが期待されます。 |
| **盲導犬受け入れの意識** | 盲導犬協会の調査では、盲導犬の受け入れに前向きな回答が6割に達したことが報告されています [4]。 | 心理的な障壁（Will）は低減しているものの、物理的な対応（Can）を支える具体的なサービスガイドラインの普及と、それに基づく環境整備が待たれます。 |

## 3. PEST分析：DEI領域
| 区分 | 分析内容 |
| :--- | :--- |
| **Politics（政治）** | 政府による「住生活基本計画」の決定や、就職氷河期世代への集中支援など、公的扶助の枠組みが具体化しており、政策的な推進が見られます [3]。 |
| **Economy（経済）** | 就労支援事業所の経営悪化が顕在化しており、福祉サービスの経済的持続可能性が問われています。これは、持続可能なビジネスモデルの構築が急務であることを示唆しています [2]。 |
| **Society（社会）** | 特別支援教育への需要増大や、盲導犬等への理解浸透など、マイノリティの権利に対する社会の期待値が高まっています。社会全体の意識変革が進んでいると言えるでしょう [1] [4]。 |
| **Technology（技術）** | 物理的なリソース不足（教室等）を補完するための、エデュテックやリモート支援技術の実装が急務です。テクノロジーを活用した新たなサービス提供モデルの検討が求められます [1]。 |

## 4. 総括
本日のリサーチから、DEI領域におけるサービスデザインの焦点は「アクセスの確保」から**「質の担保と持続性」**へと移行していることが明確になりました。特に、制度が現場の疲弊によって形骸化している現状に対し、サービスデザイナーは**「制度と実務のギャップを埋めるサービスプロトタイプ」**を提示し、より実効性のあるソリューションを創出する役割を担うべきです。
